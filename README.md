# EldenForge


```
conda activate elden_ring_rag
```

### Front-end
```
cd elden-ring-app
npm install
npm run start
```

### Back-end
```
cd ../elden-ring-backend
npm install express cors multer
node server.js
``` 

### Database
```
cd ../elden-ring-rag
python server_launcher/launch_server.py
```

Go to `http://localhost:3000/` to see the app.
